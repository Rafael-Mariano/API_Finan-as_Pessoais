# API de Finanças Pessoais

A API deste projeto de Finanças Pessoais conta com funcionalidades
básicas para criar diferentes tipos de contas bancárias e fazer o seu
gerenciamento através dos métodos de depósito, saque e tranferência de 
valores entre contas. Tudo é registrado em histórico, que futuramente
nos ajudará na criação dos extratos, funcionalidade até então ainda não
desenvolvida.

[x] Depositar valores
